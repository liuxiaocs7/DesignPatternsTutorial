## principles：
- demo1 开闭原则
- demo2 里氏替换原则
  - `before` 改进之前的代码
  - `after` 改进之后的代码
- demo3 依赖倒转原则
  - `before` 改进之前的代码：依赖于具体的实现
  - `after` 改进之后的代码：依赖于抽象
- demo4 接口隔离原则
  - `before` 改进之前的代码：一个接口中有多个方法
  - `after` 改进之后的代码：每个无关方法一个接口便于扩展
- demo5 迪米特原则
  - `before` 改进之前的代码：一个接口中有多个方法
  - `after` 改进之后的代码：每个无关方法一个接口便于扩展
